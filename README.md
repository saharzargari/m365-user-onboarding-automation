\# Automated User Onboarding System



This project simulates a real-world IT onboarding workflow using PowerShell.



\---



\## 🚀 Features



\- Reads new user information from a CSV file



\- Validates required fields



\- Simulates account creation



\- Simulates group assignment



\- Simulates license preparation



\- Writes logs for each action



\---



\## 🛠️ Technologies Used



\- PowerShell



\- CSV (data input)



\- File-based logging



\---



\## 🎯 Purpose



I built this project to strengthen my skills in IT automation, Microsoft 365 administration, and user lifecycle management.



This project reflects real-world scenarios handled by IT administrators, especially in environments using Microsoft Entra ID and Intune.



\---
